Каталог звезд и незвездных объектов, видимых в средних широтах северного полушария

Предлагаемый каталог является компиляцией нескольких астрономических каталогов и содержит наиболее интересную информацию о следующих типах объектов: яркие звезды, двойные звезды, рассеянные скопления, шаровые скопления, планетарные туманности, отражательные (околозвездные) туманности, яркие туманности, темные туманности, галактики. В настоящий момент здесь представлена WEB-версия этого каталога (с англо-латинскими обозначениями) - 59 файлов для 58 созвездий (созвездие Змеи разделено на две части). Пять южных созвездий (Col, Cae, Cen, Lup и Mic), имеющих области севернее -32o по склонению, из каталога исключены.

Вход в каталог осуществляется через сводную таблицу - список созвездий алфавитном порядке полных латинских названий и роспись количества объектов каждого типа для каждого созвездия: STARS - яркие звезды; NOTES - примечания по названиям и обозначениям ярких звезд; DOUBLE - двойные звезды; O_CL -рассеянные звездные скопления; GL_CL - шаровые звездные скопления; PL - планетарные туманности; REFL - отражающие (околозвездные) туманности; BR - яркие туманности; DARK - темные туманности; GAL - галактики. В этой таблице на названия созвездий сделаны линки на соответствующие файлы, а к количествам объектов каждого типа - на ту часть этих файлов, где начинается список для этого типа объектов. Внутри каждого созвездия объекты рассортированы по типам, в каждом типе - по номерам карт атласа "Уранометрия 2000.0", для каждой карты - по прямым восхождениям.

Каталог не содержит объектов южнее -34o по склонению и слабее 13m (для незвездных объектов) как не представляющих практического интереса для наблюдений со средним телескопом в средних широтах северного полушария. Однако информация о собственных названиях звезд и незвездных объектов всего неба, а также обо всех объектах каталогов Мессье, NGC и IC (без ограничений по склонению и блеску) можно получить с помощью поисковой системы моего учебного курса "Основы астрономии".

Двойные звезды (Double stars)

The Washington Visual Double Star Catalog, 1996.0. Worley C.E., Douglass G.G. US Naval Observatory <unpublished (1996)> В каталоге представлена информация о 7943 двойных звездах (кратные звёзды разделены на пары с общим обозначением) севернее -28o по склонению. Предельная звездная величина компонент и максимальное угловое разделение зависит от склонения. Для del > -15o оставлены двойные и кратные звёзды, с компонентой А не слабее 9m.5 и остальными - не слабее 12m. Для простых двойных звёэд с m1 < 6m.0 оставлены пары с разделением r < 100"; при m1 >= 6.0 ограничен блеск второй компоненты до m2 < 10m.0 и разделение до r < 30"; для -28o < del < -15o оставлены только пары с m2 < 9m.0. Для Полярной звезды приведены все компоненты, независимо от блеска.

    Cp - Идентификация компоненты, если их больше двух (кратная звезда).

    T1,T2 - Годы первого и последнего удовлетворительного наблюдения.

    P1, P2 - Позиционные углы для Т1 и Т2.

    r1, r2 - Угловое разделение для Т1 и Т2, угл. сек. В случаях, когда перемещение компонент не было обнаружено, указано только одно значение позиционного угла и разделения.

    m1, m2 - Блеск первой и второй компонент.

    Sp1,2 - Спектральный класс главной компоненты или обеих, если позволяет разделения.

    n_alf - Собственное движение по alf, "/1000 лет. Отнесено к большому кругу небесной сферы. Указано значение 15ma*cos(del), где ma - в секундах времени. Знак "+" показывает движение на восток, "-" - на запад.

    n_del - Собственное движение по del, "/1000 лет. Знак "+" показывает движение на север, "-" - на юг.

    N - Если стоит буква "О", то вычислена орбита. Когда орбитальное движение пары превышает 360o, позиционный угол и разделение компонент не указываются. Орбиты можно найти в "Fourth Catalog of Orbits of Visual Binary Stars" (Worley and Heintz, 1983), и в информационных циркулярах, публикуемых Комиссией 26 МАС.

    Lable - Обозначение звезды.
